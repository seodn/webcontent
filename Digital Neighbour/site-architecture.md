## 5. Internal Linking Plan

To establish clear semantic context and prevent keyword cannibalisation, internal links must follow these rules:

1.  **Homepage links to Locations and Industries:** The homepage (`/`) will link directly to `/locations/auckland/`, `/locations/wellington/`, etc., using precise city anchor text. It will link to industry pages using target niche anchors.
2.  **Location Pages link to Services:** Each location page (such as `/locations/auckland/`) will link back to the national core service subpages (such as `/seo/local-seo/` and `/website-development/wordpress-website-development/`) to pass authority.
3.  **Industry Pages link to Core Services:** Niche pages (such as `/industries/dental-marketing/`) will link back to `/seo/local-seo/` and `/google-ads/google-search-ads/` to show execution authority.
4.  **No Christchurch Location Page:** Links pointing to Christchurch local services must resolve to the homepage (`/`).
